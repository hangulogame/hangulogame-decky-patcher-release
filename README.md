# 한글로게임 Patcher

스팀덱에서 한국어 패치를 원클릭으로 설치하고 관리하는 [Decky Loader](https://decky.xyz/) 플러그인입니다.

## 소개

스팀덱(Steam Deck)이나 기타 SteamOS 기기에서 Steam 게임의 한국어 패치를 간편하게 설치할 수 있습니다. 게임 상세 페이지에서 한국어 패치 여부를 바로 확인하고, 버튼 하나로 다운로드부터 설치까지 자동으로 처리됩니다.

### 주요 기능

- **게임 페이지 뱃지** — 게임 상세 페이지에 한국어 패치 상태가 표시됩니다
- **원클릭 설치** — 다운로드, 압축 해제, 파일 복사까지 자동 처리
- **설치된 패치 관리** — Quick Access 메뉴에서 설치된 패치 확인, 제거, 업데이트
- **자동 언어/베타 브랜치 설정** — 패치에 필요한 게임 설정을 자동으로 변경
- **원본 파일 백업** — 패치 설치 전 원본 파일을 백업하여 안전하게 제거 가능

### 뱃지 상태

| 색상 | 상태 | 설명 |
|------|------|------|
| 회색 | 한국어 패치 있음 | 패치는 있지만 게임이 설치되지 않음 |
| 파랑 | 한국어 패치 설치 | 설치 가능한 패치가 있음 |
| 초록 | 한국어 패치 적용됨 | 패치가 설치된 상태 |
| 주황 | 한국어 패치 업데이트 | 새 버전의 패치가 있음 |

## 설치 방법

### 사전 요구사항

- SteamOS 기기 (Steam Deck, Steam Machine 등)
- [Decky Loader](https://decky.xyz/) 설치 완료
- 인터넷 연결

### 플러그인 설치

1. 스팀덱을 **데스크톱 모드**로 전환합니다
2. 아래 버튼을 클릭하여 설치 파일을 다운로드합니다

   [![다운로드](https://img.shields.io/badge/📥_HangulogamePatcher.desktop-다운로드-blue?style=for-the-badge)](https://github.com/hangulogame/hangulogame-decky-patcher-release/releases/download/installer/HangulogamePatcher.desktop)
3. 다운로드한 **HangulogamePatcher.desktop** 파일을 더블클릭합니다
4. 비밀번호 입력 창이 나타나면 비밀번호를 입력합니다
5. 자동으로 플러그인이 설치되고 완료 알림이 표시됩니다
6. 게임 모드로 돌아가면 Quick Access 메뉴에 **한글로게임 Patcher**가 나타납니다

> 이후 플러그인 업데이트는 Quick Access 메뉴에서 자동으로 안내됩니다.

## 사용 방법

### 한국어 패치 설치

1. Steam 라이브러리에서 게임의 **상세 페이지**를 엽니다
2. 게임 정보 영역 하단에 **한국어 패치 뱃지**가 표시됩니다
3. 파란색 뱃지(설치 가능)를 클릭하면 **패치 정보 팝업**이 열립니다
4. 패치 정보와 공지사항을 확인한 후 **"동의 및 설치"**를 누릅니다
5. 다운로드와 설치가 자동으로 진행되며, 프로그레스바로 진행 상황을 확인할 수 있습니다
6. 설치 완료 후 뱃지가 초록색으로 변경됩니다

### 베타 브랜치 변경이 필요한 경우

일부 패치는 특정 베타 브랜치에서만 동작합니다. 이 경우:

1. 설치 버튼을 누르면 **베타 브랜치 변경 안내**가 표시됩니다
2. 확인을 누르면 게임 설정이 자동으로 변경되고 **Steam이 재시작**됩니다
3. 게임 업데이트가 완료된 후 다시 패치를 설치합니다

### 언어 설정 자동 변경

한국어 패치가 특정 언어 설정을 요구하는 경우:

1. 패치 설치 완료 후 **언어 변경 안내**가 표시됩니다
2. 확인을 누르면 게임 언어가 자동으로 변경되고 **Steam이 재시작**됩니다
3. 패치 제거 시 원래 언어 설정으로 자동 복원됩니다

### 설치된 패치 관리

1. **Quick Access 메뉴** (···버튼)를 열고 **한글로게임 Patcher** 탭으로 이동합니다
2. **설치된 한국어 패치** 목록에서 관리할 패치를 선택합니다
3. **제거** — 패치를 삭제하고 원본 파일을 복원합니다
4. **업데이트** — 주황색 표시가 있으면 새 버전으로 업데이트할 수 있습니다

### 설치 가능한 패치 확인

1. Quick Access 메뉴의 **한글로게임 Patcher**에서 설치 가능한 패치 목록을 확인합니다
2. 스팀덱에 설치된 게임 중 한국어 패치가 있는 게임만 표시됩니다
3. **"패치 목록 업데이트"** 버튼으로 최신 패치 목록을 가져올 수 있습니다

### 패치 상세 정보

패치 정보 팝업에서 **"자세히 보기"**를 누르면 Steam 내장 브라우저로 [한글로게임](https://www.hangulogame.com) 웹사이트의 패치 상세 페이지가 열립니다.

## 지원 범위

- **플랫폼**: Steam 게임만 지원 (SteamOS에서 실행)
- 패치 목록은 [한글로게임](https://www.hangulogame.com)에서 관리되는 패치 중 호환이 확인된 것만 제공됩니다

## 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| 뱃지가 표시되지 않음 | Quick Access에서 "패치 목록 업데이트"를 눌러 메타데이터를 갱신합니다 |
| 다운로드가 실패함 | 인터넷 연결을 확인하고 다시 시도합니다 |
| 설치 후 게임이 정상 동작하지 않음 | Quick Access에서 해당 패치를 제거하면 원본 파일이 복원됩니다 |
| 베타 브랜치 변경 후 패치 설치 불가 | 게임 업데이트가 완료될 때까지 기다린 후 다시 시도합니다 |
